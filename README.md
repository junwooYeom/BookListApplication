# bookListApplication
## 사용 기술 스택

### 프로젝트 구성
- MVVM 프레임워크
- Android Clean Architecture
### 의존성 주입
- Hilt
### 네트워크
- Retrofit2
### 비동기 처리
- Kotlin Flow
- Kotlin Coroutine
### 이미지 리소스
- Glide
### 뷰 관련 라이브러리
- Android Navigation

## 프로젝트 설치 및 빌드 방법
> Android Studio 가 설치되어 있지 않다면 설치 부탁드립니다.
### 코드 확인 및 실행 시
1. 전송한 .zip 파일 압축 풀기
2. Android Studio 실행
3. Open 후 압축 풀기 된 폴더의 루트 위치 열기
4. Gradle Sync 후 Run 버튼 클릭 ( 에뮬레이터 혹은 실 기기 와 연결되어 있어야 함. )
